## University Course Ontology
 
## Description:
This project represents a simple ontology for a university academic system using Python.
 
## Ontology Concepts:
- Students
- Lecturers
- Courses
- Departments
- Classrooms
 
## Relationships:
- Student enrolled in Course
- Lecturer teaches Course
- Course belongs to Department
- Course has prerequisite
- Course taught in Classroom
 
## How to Run:
python ontology_lab.py
